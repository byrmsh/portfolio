Deneme Grafik is an academic counseling tool. Students can see score trends over time, while admins manage test and profile data from a separate panel.

The core model maps schools, classes, tests, and results so averages and comparisons stay correct across different datasets.

I built it to answer a very practical question for exam prep: not just "what score did I get?", but "am I improving?".
