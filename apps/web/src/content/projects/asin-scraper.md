This freelance project was specifically about Amazon ASIN workflows. The goal was to track and review Amazon product data in a structured panel instead of raw scripts and spreadsheets.

The web app provides role-based access, grid-heavy admin pages, and backend routes for managing ASIN-linked product records. The Prisma schema includes product history fields and HTML log storage to support scraping/debug cycles.

It was delivered as a practical internal tool for ongoing Amazon data operations.
