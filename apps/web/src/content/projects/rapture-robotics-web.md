I worked on this during my Rapture Robotics period as Programming Lead. The website was the public face of our team while we were participating in events like Inegol Tech Festival, Teknofest, THY Travel Hackathon, METU Robot Days (ODTU Robot Gunleri), and RoboBoat.

Beyond publishing content, it helped us present the team professionally to sponsors and local institutions. We even had meetings with the Mayor and Governor that resulted in support for the team.

So for me this project is directly tied to competition work, not only web development.
